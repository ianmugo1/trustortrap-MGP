# TrustOrTrap 

A gamified cyber-awareness web app built with:

- **Frontend:** Next.js 14 + TailwindCSS
- **Backend:** Express + MongoDB
- **Runtime:** Node 20
- **Repo Type:** Monorepo (client + server)

---

##  Local Dev
```bash
npm i
npm run dev      # runs both server (:5000) and client (:3000)
